# 采购收货管理 — 20步流程推演

## 用户操作链 (20步)

```
登录 → 菜单: 供应链与成本 → 采购收货
  → 1. 收货列表页 (表格: 收货单号/供应商/PO单号/日期/状态/金额/操作)
  → 2. 搜索/筛选 (单号/供应商/PO/日期范围/状态)
  → 3. 新建收货 → 选择PO(弹出PO选择器) → 自动加载PO明细
  → 4. 在PO明细上输入收货数量(默认=PO数量)
  → 5. 填写仓库/库位/备注 → 保存 → 状态: pending_inspection
  → 6. PO状态自动更新: ordered→部分收货/全部收货
  → 7. 来料检验 → 选择收货单 → 输入抽检数量
  → 8. 逐项填写: 合格数/不合格数/缺陷描述
  → 9. 判定: 合格(Pass)/让步接收(Concession)/退货(Reject)
  → 10. 检验完成 → 收货单状态更新
  → 11. 退货流程: 生成退货通知 → 通知供应商
  → 12. 查看收货详情: 收货信息+质检结果
  → 13. 库存更新: 合格品自动入库 (预留)
  → 14. 供应商绩效: 退货率自动计算 (预留)
  → 15. 质检统计: 批次合格率趋势 (预留)
  → 16. 权限: 采购可收货/质检可检验/只读查看
  → 17. 发票匹配(预留)
  → 18. 按需生成入库单(预留)
  → 19. 关联应付账款(预留)
  → 20. 供应商评分自动联动(预留)
```
