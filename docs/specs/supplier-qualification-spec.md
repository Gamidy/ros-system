# 供应商准入管理 — 20步流程推演

## 用户操作链 (20步)

```
注册/登录
  → 菜单: 供应链与成本 → 采购管理
  → 1. 供应商列表页 (表格: 编号/名称/品类/状态/评分/联系人/操作)
  → 2. 搜索/筛选 (名称/品类/状态)
  → 3. 新增供应商 → 弹窗: 名称/编码/品类/联系人/手机/邮箱/地址 → 保存
  → 4. 编辑供应商 → 弹窗预填 → 保存
  → 5. 查看供应商详情 → 基本信息 + 评估记录
  → 6. 供应商评估 → 选择评估维度(品质/交期/成本/服务) → 打分(1-100)
  → 7. 权重配置表 (各维度权重)
  → 8. 综合评分自动计算 (加权平均)
  → 9. 供应商状态切换 (潜在/准入/合格/不合格/暂停)
  → 10. 删除供应商 (软删除)
  → 11. 采购管理首页: KPI卡片 (供应商总数/合格数/平均分/低分预警)
  → 12. 供应商排名 (按综合评分)
  → 13. 评估历史趋势 (折线图)
  → 14. 品类分布 (饼图)
  → 15. 导出供应商清单 (Excel)
  → 16. 权限控制: 采购可管理, 品质可评估, 其他只读
  → 17. 关联BOM物料: 物料指定首选供应商
  → 18. 供应商物料报价管理 (待扩展)
  → 19. 采购订单生成 (待扩展)
  → 20. 到货质检关联 (待扩展)
```
