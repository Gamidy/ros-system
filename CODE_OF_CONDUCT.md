# ROS Code of Conduct

## 我们的承诺

作为 ROS 社区的参与者，我们承诺为所有参与者创造一个开放、包容、尊重的环境，无论其经验水平、身份或背景如何。

## 我们的标准

**受鼓励的行为：**

- 尊重不同的观点和经验
- 建设性的反馈和开放的讨论
- 关注什么是平台最好的，而非什么是个人的偏好
- 对社区其他成员展现同理心
- 遵守 Constitution 第十二条 — Engineering Truth 永远凌驾于观点之上

**不被接受的行为：**

- 人身攻击或贬低性评论
- 无视 Constitution 和 Foundation 的讨论或决策
- 绕过 Architecture Review 的代码变更
- 将个人偏好置于平台架构一致性之上

## 执行

违反本行为准则的行为应向 Architecture Board 报告。
Board 有权采取任何其认为适当的行动，包括要求停止行为或永久移除参与资格。
