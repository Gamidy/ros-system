# 欧盟9K变频空调策划 - 问题记录与修复

## 已修复问题

### 1. 前端路由权限bug（已修复 ✅）
- **问题**: 点击策划列表「详情」按钮导航到 `/product-plans/:id` 时，前端路由守卫 `hasRouteAccess` 做精确路径匹配，导致动态路由被拦截并重定向到驾驶舱。
- **原因**: `hasRouteAccess` 只检查 `allowedPaths.has(path)` 精确匹配，`/product-plans/xxx` 不在白名单中。
- **修复**: 在 `frontend/src/stores/auth.ts` 的 `hasRouteAccess` 中添加前缀匹配逻辑：如果路径以任何允许路径+`/`开头，则放行。
- **部署**: 重新构建前端并部署到服务器 `/opt/ros-system/frontend/dist/`，添加了 `?v=2` 缓存清除参数。

### 2. 前端缓存策略导致部署不生效
- **问题**: Nginx对静态文件没有配置缓存策略，浏览器缓存旧JS文件导致修复代码不生效。
- **修复**: 修改 `index.html` 添加 `?v=2` 缓存破坏参数到JS/CSS资源路径。

## 待修复问题

### 3. 后端缺少制冷剂(refrigerant)必填校验 🔴
- **问题**: 产品策划表单中「制冷剂」字段为空也能保存成功，说明后端 `update_plan` 接口没有对 `refrigerant` 做必填校验。
- **涉及文件**: `backend/app/api/product_plan_crud.py` 或相关schema
- **建议**: 在 `PlanUpdate` schema中将 `refrigerant` 改为必填字段，或在路由处理器中添加非空验证。

### 4. 推进流程按钮无响应 🔴
- **问题**: 点击详情页底部「推进流程」按钮后，页面没有变化，未显示成功/失败反馈。
- **可能原因**: 后端 `/api/product-plans/{id}/advance` 端点可能返回错误但前端未正确处理。

### 5. 复盘记录不存在警告 ⚠️
- **问题**: 详情页底部持续显示 "Not Found / 复盘记录不存在" 错误信息。
- **可能原因**: 复盘模块尚未初始化，需要在推进流程到复盘阶段后创建复盘记录。

### 6. 前端会话过期后无优雅处理 ⚠️
- **问题**: 会话过期后直接跳转到登录页，填写的表单数据丢失。
- **建议**: 添加自动保存草稿功能或在会话恢复后恢复未保存的数据。
